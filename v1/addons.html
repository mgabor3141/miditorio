Addons

Addons can make your songs flashier – quite literally. The blueprints you'll find here can be added to any MIDItorio song. If you have any questions, or would like to submit your own addon, send a message here!

Time Display by mentlerd

Displays elapsed time. Connect the red wire from the song memory to the red wire on the pole.

Bigger Time Display by mentlerd

For when size matters. A display with slightly larger numbers. Connect the red wire from the song memory to the red wire on the pole.

Piano Visualizer by Tritex

4 octave piano-like lightshow. Connect the combinators on the left side (can be moved for convenience) to the green wire that goes to the speakers. The first column of combinators are filters, set the signals for the instruments you want to include in the display. Keys can be individually coloured through the decider combinator above them.
